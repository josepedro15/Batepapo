# CRM WhatsApp Pro

Sistema SaaS Multi-tenant de gestão de atendimento via WhatsApp.

## Stack
- Next.js 14 (App Router, TypeScript)
- Supabase (PostgreSQL + Auth + Realtime)
- TailwindCSS v4 + Radix UI
- Vercel (Deploy)

## Features
✅ Multi-tenancy (N:N User-Organization)  
✅ Kanban CRM (Drag-and-Drop)  
✅ Contact Center (Chat Realtime)  
✅ API Pública (Integração n8n)  
✅ Campanhas em Massa (Queue System)  

## Quick Start

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000
